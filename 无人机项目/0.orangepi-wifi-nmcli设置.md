##  nmcli
> [linux nmcli](https://www.xyan666.com/posts/1165531937/)

 查看当前可用 wifi
```shell
nmcli device wifi list
```
连接 wifi
```
sudo nmcli device wifi connect <wifi名> password <密码>
例：
sudo nmcli device wifi connect H3C  password   # 没有密码的wifi
sudo nmcli device wifi connect 408_luyouqi  password 408408408 # 有密码的wifi
```

查看网络链接状态
```
nmcli connection show
```

设置连接优先级
```
nmcli connection modify <wifi名> connection.autoconnect-priority <n> # 优先级默认为 0，正数优先级高，负数低；一个非负整数n 0~255？
例：
nmcli connection modify DAWN connection.autoconnect-priority 10
nmcli connection modify DAWN connection.autoconnect yes
nmcli connection show DAWN |grep priority
nmcli connection show DAWN |grep autoconnect

查看优先级
nmcli connection show <wifi名> |grep priority
```



关闭某连接
```
nmcli connection down wifiname
```
 启动某连接
```
nmcli connection up wifiname
```

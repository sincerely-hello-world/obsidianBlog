
标准化  `node.get_logger().warn/info`的输出格式：
```bash
export RCUTILS_CONSOLE_OUTPUT_FORMAT="[{file_name}:{line_number}:{function_name}:{name}:{message}"

export RCUTILS_CONSOLE_OUTPUT_FORMAT="[{line_number}:{function_name}:{name}:{message}"
 
export RCUTILS_CONSOLE_OUTPUT_FORMAT="[{severity} {time}] [{name}]: {message} ({function_name}() at {file_name}:{line_number})"
ros2 run logging_demo logging_demo_main
```

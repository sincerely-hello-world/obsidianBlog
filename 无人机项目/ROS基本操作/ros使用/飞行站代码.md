## 开机启动方案
```
orangepi@orangepi:~/Desktop/UAV_CAR__Test/UAV_CAR/src$ cat /etc/systemd/system/uav_car.service

[Unit]
Description=/etc/uav_car.local Compatibility 
ConditionPathExists=/etc/uav_car.local
 
[Service]
Type=forking
ExecStartPre=/bin/sleep 3 
ExecStart=/etc/uav_car.local start 
TimeoutSec=0 
StandardOutput=tty 
RemainAfterExit=yes 
 
[Install]
WantedBy=multi-user.target

```

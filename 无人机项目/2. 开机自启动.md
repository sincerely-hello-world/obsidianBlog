## 飞行站 uav_car的开机自启
在对应板卡上操作：
vi ~/.config/autostart/mycli.desktop

```bash
[Desktop Entry]
Type=Application
Exec=/bin/bash -c 'sleep 2 && /home/orangepi/Desktop/UAV_CAR__Test/start.sh'
Hidden=false
NoDisplay=false
Terminal=true
Name=fly
StartupNotify=false
```

如果要取消，就使用如下指令：
```
rm  ~/.config/autostart/mycli.desktop
```

## 地面站 yhc-g的开机自启
在对应板卡上操作：
vi ~/.config/autostart/myui.desktop

```bash
[Desktop Entry]
Type=Application
Exec=/home/orangepi/Desktop/yhc-g/start.sh # 确保是 这个启动脚本-----------
Hidden=false
NoDisplay=false
Terminal=false # 不显示桌面终端的日志输出
Name=PyQtGUI
```

如果要取消，就使用如下指令：
```
rm  ~/.config/autostart/myui.desktop
```

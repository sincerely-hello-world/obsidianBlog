## 飞行站 uav_car的开机自启
在对应板卡上操作：
vi ~/.config/autostart/mycli.desktop

```bash
[Desktop Entry]
Type=Application
Exec=/home/orangepi/Desktop/UAV_CAR__Test/start.sh # 确保是 这个启动脚本-----------
Hidden=false
NoDisplay=yes
Terminal=true # 显示桌面终端的日志输出
Name=飞行站自启
```

如果要取消，就使用如下指令：
```
rm  ~/.config/autostart/mycli.desktop
```

## 地面站 yhc-g的开机自启
在对应板卡上操作：
vi ~/.config/autostart/myui.desktop

```bash
[Desktop Entry]
Type=Application
Exec=/home/orangepi/Desktop/yhc-g/start.sh # 确保是 这个启动脚本-----------
Hidden=false
NoDisplay=false
Terminal=false # 不显示桌面终端的日志输出
Name=PyQt交互界面
```

如果要取消，就使用如下指令：
```
rm  ~/.config/autostart/myui.desktop
```

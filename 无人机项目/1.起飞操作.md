1. mobaxterm ssh连接上飞行端， 执行启动服务
```
./start_uav.sh

systemctl stop uav_car.service
systemctl start uav_car.service 
systemctl restart uav_car.service 

cd ~/Desktop/UAV_CAR__Test/UAV_CAR/src/uav_car_launch/launch

sudo ./start_uav.sh
# 密码为 orangepi
# 需要摇晃飞机，使得t265置信度通过
```


## 用户添加

| 对比项  | `adduser` (推荐)          | `useradd` (底层工具)      |
| :--- | :---------------------- | :-------------------- |
| 本质   | 高级Perl脚本，`useradd`的友好前端 | 底层二进制命令，直接操作用户数据      |
| 交互性  | 交互式，逐步引导                | 非交互式，需手动指定所有参数        |
| 家目录  | 自动创建并初始化                | 默认不创建，需加 `-m` 参数      |
| 密码   | 交互式提示设置                 | 不设置，需额外执行 `passwd` 命令 |
| 适用场景 | 手动创建用户，简单快捷不易出错         | 需要自定义脚本，精细把控创建用户      |
|      | 用作实体用户，可以直接登录系统         | 一般用作udev硬件设备管理        |

```bash
sudo groupadd gpio
sudo groupadd pwm
sudo usermod -aG gpio,pwm $USER
groups $USER 

cat /etc/group # all groups
id # check user's  group 
```
